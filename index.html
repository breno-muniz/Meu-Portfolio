Tecnologias

Javascript

JavaScript é uma linguagem de programação que permite a você implementar itens complexos em páginas web

ReactJs

O ReactJS é uma biblioteca de JavaScript desenvolvida pelo Facebook, utilizada para construir interfaces de usuário.

    SASS

    SASS é uma ferramenta que facilita a criação de estilos para páginas web, ele permite o uso de recursos como variáveis e funções, que ajudam a reutilizar e modificar o código facilmente.

  • HTML5

    HTML é a linguagem usada para criar páginas na internet. Ele organiza o conteúdo como texto, imagens e links, definindo a estrutura do site.

    CSS3

    CSS é uma linguagem usada para definir o visual de páginas web, como cores, fontes, e layout. Ele permite que os desenvolvedores estilizem o conteúdo do site e criando designs atraentes e consistentes.
